Selector Name Description Syntax Example

Basic Selectors

Universal Selector Selects all elements. Optionally, it may be restricted to a specific namespace or to all namespaces.

* ns|* *|*

*

will match all the elements of the document.
Type Selector Selects all elements that have the given node name.

elementname

input

will match any

<input>

the elements of the document.
Class Selector Selects all elements that have the given

class

attribute.

.classname

.index

will match any element that has the class of "index".
ID Selector Selects an element based on the value of its

id

attribute. There should be only one element with a given ID in a document.

#idname

#toc

will match the element that has the ID "toc".
Attribute Selector Selects all elements that have the given attribute.

[attr] [attr=value] [attr~=value] [attr|=value] [attr^=value] [attr$=value] [attr*=value]

[autoplay]

will match all elements that have the

autoplay

attribute set (to any value).

Grouping Selector

Selector List The

,

div, span

will match both

<span>

and

<div>

elements.

Combinators

Descendent Combinator The

" "

(space) combinator selects nodes that are descendants of the first element.

A B

Child Combinator The
